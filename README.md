#Клиент-серверное чат-приложение (Java Socket Programming)

#Выполнили:
Коростелев С., Литвинов А., Чудова А., КБ-241.

#Описание:
Это простое интерактивное многопоточное клиент-серверное чат-приложение, разработанное на Java. Приложение позволяет нескольким клиентам подключаться к серверу и обмениваться сообщениями друг с другом в реальном времени.

#Запуск приложения
1. Склонируйте этот репозиторий на свой локальный компьютер и откройте проект в предпочитаемой вами IDE или просто перейдите в папку проекта из командной строки.
2. Запустите сервер, выполнив файл Server.java, и следуйте инструкциям.
3. Запустите клиент, выполнив файл Client.java. Вы можете запустить столько клиентов, сколько захотите. Введите тот же номер порта сервера, когда будет предложено на стороне клиента.
4. После подключения к серверу вы можете начать отправлять сообщения другим подключенным клиентам.
